Installation & Setup

1️⃣ Clone the Repository

git clone https://github.com/your-username/your-repo.git
cd your-repo

2️⃣ Install Dependencies

yarn install

Or with npm:

npm install

3️⃣ Run the Project

➤ Run Web (Next.js)

yarn dev

➤ Run iOS / Android (Expo)

expo start

Note: Ensure you have Xcode (for iOS) or Android Studio (for Android) installed.
